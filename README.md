### BeautifulSoup, Selenium을 활용한 야놀자, 네이버 크롤링 및 야놀자 크롤링 데이터 시각화

### 프로젝트 개요

1) 크롤링 라이브러리를 활용해 야놀자 사이트의 평일/불금 숙박 시설 가격, 네이버 부동산 가격 크롤링

2) 야놀자 크롤링 데이터의 경우 Tableau를 활용해 서울 구별 평일 및 불금의 숙박비, 대실비 지도 시각화
